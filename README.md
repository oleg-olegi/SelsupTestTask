# CrptApi

Класс для работы с API Честного знака, реализованный на языке Java (версия 17). Класс является thread-safe и поддерживает ограничение на количество запросов к API.

Так как реальный ресурс по адресу https://ismp.crpt.ru/api/v3/lk/documents/create недоступен, в классе CrptApi реализовано мокирование ответа для тестирования. Это позволяет тестировать функциональность без необходимости реальной авторизации и отправки запросов к недоступному ресурсу.

## Описание

Класс `CrptApi` предоставляет метод для создания документа для ввода в оборот товара, произведенного в РФ.
